# PHP Code Standards Skill

Ensures PHP code adheres to PSR-1, PSR-4, PSR-12 standards and MikoPBX conventions.

## Key Features

- **Full PSR Compliance**: PSR-1, PSR-4, PSR-12, PSR-3, PSR-11
- **Type Safety**: Strict types, full type declarations
- **Modern PHP 8.3**: Enums, match, readonly, typed constants
- **Quality Tools**: PHPStan, PHP CS Fixer automation
- **Code Examples**: Real-world patterns and templates
- **Validation Workflow**: Pre-commit checks and automated formatting

## Quick Usage

1. Use when writing new PHP code or refactoring
2. Follow file structure template with GPL license header
3. Add `declare(strict_types=1);` after opening tag
4. Declare types on all properties, parameters, and returns
5. Run `composer cs-fix` before committing
6. Verify with `vendor/bin/phpstan analyse file.php`
